Instalação

1. Clonar repositório do Github
https://github.com/danilomeneghel/prova_php.git

2. Importar o arquivo "prova_php.sql" para o Banco de Dados

3. Configurar Banco de Dados 

Abra o arquivo "/questao4/app/Config/database.php" e altere as variáveis conforme o banco de dados local

class DATABASE_CONFIG {

	public $default = array(
		'datasource' => 'Database/Mysql',
		'persistent' => false,
		'host' => 'localhost',
		'login' => 'root',
		'password' => '',
		'database' => 'prova_php',
		'encoding' => 'utf8',
	);
}

4. Rodar o projeto da pasta "questao4".

Exemplo:
http://localhost/prova_php/questao4/


Requisitos:

Caso o CSS e JS não carregue, será necessário habilitar o mod_rewrite no Apache
Abra o arquivo conf/httpd.conf dentro do seu Apache e descomente a seguinte linha:
LoadModule rewrite_module modules/mod_rewrite.so

Após reinicie o Apache.
Isso fará com que o mod_write seja habilitado, que é o responsável pelo carregamento de URLs amigaveis.
